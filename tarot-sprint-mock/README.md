# Tarot Sprint — Mock Web App

A simple static mockup for a fast‑learning tarot app with monetization tiers.

## Run

- Features and onboarding steps
- Pricing tiers (Free / Pro / Master)
- Demo learning screen (card of the day, SRS queue, streaks)
